# Leetspeak Converter

This is my first program in which I make use of the command line argument array `argv`. After a couple of checks that determine if the program is being used properly, the argument the user provides gets transformed into its leetspeak variant.

In this project I got to use a few new libraries I hadn't used before, mainly `string.h` and `ctype.h`, both very useful for string manipulation. As well as this, I implemented a very convenient `switch` statement for the majority of letters in the alphabet that would be converted to their leetspeak symbols.

Here is the list of letters and their leetspeak counterparts:

- A --> 4
- B --> 8
- C --> [
- D --> )
- E --> 3
- G --> 6
- I --> 1
- O --> 0
- S --> 5
- T --> 7
- Z --> 2
